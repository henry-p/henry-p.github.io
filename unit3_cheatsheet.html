Cheatsheet

Sublime Text

í
In this cheatsheet I give an overview of useful commands in Sublime Text (2) on a Mac. It is written from the perspective of someone who is new to Ruby, Web Development and Sublime, so they're easy to follow. First, I share my configuration. Second, my favourite commands that I use all the time. Third, a more complete compilation of commands which I borrwed from cheatography.com (source: http://www.cheatography.com/martinprins/cheat-sheets/sublime-text-3-osx/).

Configuration:
- Theme: When the working environment is dark, I use a modified version of Monokai, when it is bright, I use Solarized (Light).
- Package Control, it's super useful for installing new packages.
- Packages: AdvancedNewFile, AutoFileName, BracketHighlighter, Colorsublime-Themes, DocBlockr, Emmet, Gist, Hayaku, Indent XML, Line Endings Unify, LiveReload, Markdown Preview, Prefixr, PxToEm, PyV8, Restart, RubyFormat, Schemr, SideBarEnhancement, SublimeLinter
The bold printed packages are particularly useful.
- I sometimes use Sublime's Vintage Mode, but you would have to dive into Vim relatively deep, in order to have a strong benefit from it.

Favourite Shortcuts:
- You should be familiar with Regular Expressions (Regex). It's dry, it's not fun, it's quick to learn. Trust me, you'll save a lot of time knowing some basic expressions!
- CMD + D | Find next occurence of selection.
- CMD + K + B | Toggle Sidebar.
- CMD + SHIFT + P | Command Palette.
- CMD + SHIFT + R | Reformat Ruby Code (RubyFormat is needed).
- CMD + SHIFT + P, "rein..." -> "Reindent Lines" | Reindent Code.
- CTRL + TAB | Switch to next open tab.
- CMD + SHIFT + L | When several lines are highlighted, make a cursor on every line.
- CMD + -> | Jump to end of line. Also useful with SHIFT (highlighting)..
- CMD + J | Join several lines to one.
- CMD + ALT + F | Find and replace.
- ESC | Close context menu.
- CMD + W | Close tab.
- CMD + / | Toggle comment.
- CMD + N | Create a new file in current directory.
